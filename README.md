# Real-Time Face & Eye Detection System using OpenCV

## Overview

This project is a real-time computer vision application developed using Python and OpenCV. The system detects human faces and eyes through live webcam video streaming using Haar Cascade classifiers.

The application performs live frame processing, grayscale conversion, face localization, and eye detection, making it suitable for AI-based monitoring and surveillance applications.

---

## Features

- Real-time webcam face detection
- Eye detection within detected face regions
- Live video stream processing
- Haar Cascade-based object detection
- Bounding box visualization
- Simple and lightweight implementation

---

## Technologies Used

- Python
- OpenCV
- Haar Cascade Classifier
- Computer Vision
